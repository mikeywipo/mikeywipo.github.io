Simple problem...

There was this one time at course..that I was trying to reference to some information in a list. It was just a bit list of random test scores for four students, the first thing we had to do was sort it into smaller groups which was all good, we made everyone their own thing with their results, all good . Then we had to make another function that would add to the list. So I write the code for add a score all good then it's time to tell it where to put the score and I'm referencing the the list, then inside that list I reference to ths students name, sweet all good. Then I try run the function. No good. I go back over the code to check that I've typed everything correctly and I had so I try run it again. No good. 30 minutes pass and during that time I've tried rereading the code a thousand times and even though I was taking regular breaks I felt frustrated as! So I look at the solution...I forgot to reference it to the smaller list we made for each person. Doh! So what I learnt was it's not always the syntax in the code thats wrong, open your mind to looking at all possible problems and try to look at it from a different angle.
