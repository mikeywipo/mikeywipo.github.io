Explain control flow and loops using an example process from everyday life

Think of control flow as the order of how tasks are actioned kind of like when you're leaving the house and you need to set the alarm. If you really think about it, there is an order to what needs to be done first. Like set the alarm then lock the door, it can't really be done the other way around and Loops are things that repeat over and over and an example of this is the sunrise and sunset. Everyday the sun comes up and goes down, then the next day comes and it rises up and goes down.

Describe what the DOM is and an example of how you might interact with it.

When an html document is loaded, a browser will create a copy of everything in that html document for us to interact with. The new copy is made up of nodes and is called the DOM, and it can be changed and customized however you want without impacting the original html file. It's also useful to test how things might look on the page without changing the code.

Explain the difference between accessing data from arrays and objects.

When we access data from an array we use indexes [] and when we access data from an object we have a few options we can use dot notation or we can use brackets [] or we can even use a variable.

Rather than remembering all of your 'checklist' each time you can have a function to do this all automatically.
